# Notes

## Current Understanding
Atlas OS is a personal AI operating system built around:
- ChatGPT as the command center
- Do Browser as the browser operator
- GitHub as the system spine
- Gmail and Loom as communication/context tools
- a future local layer for privacy, voice, and personalization

## Build Principles
- voice-first where possible
- minimize manual typing
- reduce unnecessary clicks
- stop before irreversible action
- convert repeated work into reusable workflows
