# Project Template

## What Qualifies as a Project
A project is a larger multi-task effort with a durable outcome, multiple dependencies, or a need for ongoing coordination.

## How Projects Relate to Tasks
Projects contain or connect multiple tasks. Tasks are the working units. Projects provide the bigger-picture outcome, scope, and milestones.
