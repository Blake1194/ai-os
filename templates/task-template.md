# Task Template

## Usage Guidance
Use this template when a request needs a tracked outcome, clear ownership, step-by-step progress, or review. Do not create a task for pure noise, temporary thoughts with no action, or duplicate work already tracked elsewhere.

## When to Create a Task
Create a task when:
- there is a clear objective
- action is required
- the work may span more than one step
- approval boundaries matter
- progress should be logged

## When Not to Create a Task
Do not create a task when:
- the input is just reference material
- the item belongs inside an existing task
- the request should become only a note, decision, or deferred idea
