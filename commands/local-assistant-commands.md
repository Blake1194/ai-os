# Local Assistant Commands

## Purpose
These commands help define, review, and grow the local assistant effort inside Atlas OS.

## Define Assistant Scope
- "Define the scope of my local assistant and separate must-have features from later features."

## Compare Local vs Cloud Responsibilities
- "Compare which assistant responsibilities should stay cloud-based for now and which should move local-first later."

## Design Next Assistant Milestone
- "Design the next milestone for my local assistant project and keep it practical."

## Log Assistant Design Decisions
- "Turn this assistant architecture choice into a decision record."

## Break Assistant Build into Tasks
- "Break the local assistant project into the next small task set using Atlas task structure."

## Review Assistant Architecture
- "Review the current local assistant architecture and tell me the next best improvement."
