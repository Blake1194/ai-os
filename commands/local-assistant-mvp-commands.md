# Local Assistant MVP Commands

## Purpose
These commands help define, narrow, compare, and operationalize the first local assistant MVP.

## Define MVP Scope
- "Define the MVP scope for my local assistant and keep only what is necessary for the first useful version."

## Narrow Features to MVP Only
- "Review this assistant feature list and remove everything that does not belong in the first MVP."

## Compare Implementation Options
- "Compare realistic implementation options for the first local assistant MVP and recommend the best one for Atlas OS right now."

## Choose First Implementation Direction
- "Choose the first implementation direction for my assistant MVP and explain why it is the best tradeoff."

## Record Out-of-Scope Items
- "Turn these assistant ideas into an explicit out-of-scope list so the MVP stays controlled."

## Define Success Criteria
- "Define measurable success criteria for the first assistant MVP so I know when it is useful enough."

## Turn Spec Into Build Tasks
- "Break this assistant MVP specification into the next small build tasks using Atlas task structure."
