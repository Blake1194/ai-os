# Command Library

## Inventory Commands
- Scan my current browser session and sort tools into core, optional, and remove-later.
- Audit my open tabs and identify active work versus noise.

## Browser Research Commands
- Research this topic, open the best sources, summarize them, and stop before saving anything.
- Compare these tools and recommend one primary option.

## File Organization Commands
- Create the Atlas folder structure and stage the files.
- Map this mounted folder into Atlas memory and tell me what matters.

## Communication Commands
- Turn this Loom into a Gmail draft and stop before send.
- Prepare a reply from this context and stop before sending.

## Project Management Commands
- Turn this task into a workflow spec.
- Turn this project idea into a tracked next-action plan.

## Automation Commands
- Build this repeated browser task as a reusable workflow and stop before final submit.
- Reduce this process into the fewest safe steps.

## Troubleshooting Commands
- Inspect this page, identify the blocker, and prepare the safest fix.
- Tell me the smallest blocker only and the exact next page or field needed.

## Memory Capture Commands
- Save this session into Atlas memory as checkpoint, summary, and next actions.
- Convert this recording into notes, commands, and workflow steps.

## Local Assistant Build Commands
- Translate this repeated task into a future local Atlas capability.
- Design the next local component for Atlas using what I already have.

## Daily Operating Commands
- Run Atlas startup: scan tabs, inbox, blockers, and next best action.
- Run Atlas shutdown: log progress, save context, and queue tomorrow's first task.
