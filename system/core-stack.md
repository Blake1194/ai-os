# Core Stack

## Chosen Core Tools
- **ChatGPT** — command center and reasoning layer
- **Do Browser** — browser operator and execution layer
- **GitHub** — system spine, memory, workflows, logs
- **Gmail** — communication hub
- **Loom** — async voice/video capture and context sharing
- **Workspace Files** — working drafts, exports, and intermediate files

## Responsibilities
- **ChatGPT** owns task understanding and routing
- **Do Browser** owns page navigation, field filling, and safe staging
- **GitHub** owns permanent structure and versioned memory
- **Gmail** owns message drafting and communication workflows
- **Loom** owns recorded instructions and context capture
- **Workspace** owns temporary working files and local staging

## Deprioritized for Now
- overlapping browser AI assistants as primary planners
- advanced local models before the core system is stable
- full automation code before one workflow is proven
