# Session Review Workflow

## Purpose
Define how each working session should be summarized and retained.

## Review Areas
- what was attempted
- what was completed
- what changed
- what decisions were made
- what is still blocked
- what the next best action is
- what should be added to the command library
- what should be added to memory
